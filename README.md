## Development setup ðŸ› 

- [Node.js and npm](https://nodejs.org) installed (LTS)
```sh
cd directory-name

# Install dependencies

# You'll need gulp-cli installed
npm i gulp-cli -g

# Start gulp and create dist folder if it doesn't exist
gulp watch
```
